Event Planner

Geronigo's event planner is a space for event organisers to finalise and complete their user journey, post enquiry or booking.

Story goals:

  1. Convert event enquiries to payments Adding value to the event organiser can remove pain points and encourage users to make payment.
  2. Engage wider audience Event participant sizes range from 2 to over 20. Engaging these guests creates brand awareness & free marketting.

Story skills:

User analysis Highlighed key pain points which restricted user payments.

Mobile app A supporting mobile app was seen as a viable means of engaging event participants.

UI redesign A mobile first redesign was required in a addition to new feature design.

UI animations UI animations and transitions were needed to support the mobile app.

UX journey & new feature design

Analysing enquiry and user feedback, 3 new features were created to improve the overall user experience.

Individual Payments

Event organisers found the task of accumilating guest payments the most prohibitive reason for completing a booking.

  • Organiser no longer needs physical payment from guests.
  • Integrated with guest list UI.
Guest Management

Connecting the event planner with social network and device contact list account allows easy access to invite guests.

  • RSVP and guest payments encourage wider app usage.
  • Simplified method compared to previous manual input & link sharing.
Instant Messaging

Answering user enquiries through an IM system reduces information delays and uncertainty.

  • General enquiries are submitted online rather than direct calls.
  • A direct marketting line is established.
