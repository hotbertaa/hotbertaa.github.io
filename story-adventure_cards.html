Design system & component library

Implemented with Figma and Zeplin due to an increase in website redesign, shared UI features and mobile application development.

Style guides and component libraries speed up the design process and better communicate visual hierachy to developers. Features within Figma, such as component variants and auto layout make design iteration much more efficient.

Style guides
  • Maintains a pre defined visual hierachy.
  • Makes communication between teams more efficient.
  • Saves time by reducing cross site variations.
Components & variants
  • Variants enables rapid design prototyping & design iteration.
  • Simplifies A/B testing.
  • Connects design and development thinking.
Auto layout
  • Creates pixel perfect layouts and removes human error.
  • Better enables style guide spacing requirements.
  • Reduces time spent on design enhancement.
