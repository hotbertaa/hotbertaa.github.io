Peter Roper

UI Designer & Web Developer

I am a versatile & adaptable UI designer. My aim with all projects is to combine a captivating design with an intuitive experience which engages a user.

My design philosophy Understanding, Inspiration, Simplicity

Understand User

Good design adapts to the users needs. A users requirements may differ from what is classed as 'good design'.

avoid design trends, unless a user demands it.

Inspire Yourself

Good design isn't a blank canvas. It comes from having a broad sample of inspiration as a start point.

reference competition, use design resources.

Simplify Needs

Good design knows when to remove something. Refine a design by taking away elements to improve the overall design.

form follows function, performance is crucial.

Prototype for Geronigo's event planner

Below is the prototyping work for a proposed 3rd iteration of the Geronigo event planner, produced using Sketch & Principle.

Pros

Beyond the visual appeal of the new redesign, putting the focus on the guest list and its use in managing individual payments has given the event user a valuable tool and reason for using the event planner, aiding its ability to attract multiple users.

Cons

With more time I would like to add further confirmation pages on changes in state such as RSVP, payment confirmation. I'd also like to further the transitioning scope with principle.
