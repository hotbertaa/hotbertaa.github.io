Every Adventure...

The worlds first international adventure network. For transport theres Uber, for food theres Deliveroo, for beds theres Airbnb. But what about for the things to do? Introducing Geronigo, 'One World... Every Adventure'.

  1. Demonstrate breadth of choice The user needs to feel like they've discovered a unique resource
  2. Simplify this choice for the user activity, location, price range, age, rating... more or less? it needed testing...

Considerations:

User personas There are a variety of different user types; Explorers, activity focussed, event planners and more.

User journeys Search engine links create different points of entry in the UX journey.

Filtering What level of filtering is needed? Category, activity, location, price range, age, rating... more or less?

Information architecture Page content needs to match screen purpose, funnel the user journey and also link to the wider network.

Solution: User Funnels
